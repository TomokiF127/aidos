# AIDOS

AI-Driven Orchestration System

## Setup

```bash
npm install
```

## PoC Commands

```bash
# PoC-1: タスク分解検証
npm run poc:task

# PoC-2: Worker Thread検証
npm run poc:worker

# PoC-3: TUI検証
npm run poc:ui

# PoC-4: 統合検証
npm run poc:all
```
